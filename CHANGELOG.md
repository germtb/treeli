# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-01-29

### Added

- Initial release of treeli
- Declarative TUI framework with JSX support
- Efficient cell-based diffing with Merkle tree optimization
- Solid.js-style reactive primitives (`createSignal`, `createEffect`, `createMemo`)
- Flexbox-inspired layout system with `direction`, `justify`, `align`, `gap`
- Built-in elements: `<box>`, `<text>`, fragments
- Input handling with focus management
- Button and Input primitives
- Border styling support (single, double, rounded, bold)
- Padding and margin support
- Absolute positioning for overlays
